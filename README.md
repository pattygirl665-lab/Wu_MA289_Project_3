**Project 3: Logistic Regression Weaknesses in High Feature, Low Observation Ratio Datasets**

In understanding a model, one needs to understand the limitations of the modeling method.
Using a synthetic dataset that is intentionally designed to stress-test the method and a real-
world dataset with similar structural challenge of little observations with multiple features,
the logistic regression model demonstrates problems from overfitting variables and poor
generalization, which is more apparent in synthetic datasets than in real-world datasets, and
high computational intensity with more features, which is more apparent in the real-world
dataset with a higher number of features than the synthetic dataset.

**Getting Started**
This Repository contains the Day 0, Day 1, Day 100, and Day 120 .csv files. To start, download the data file to then be able to mount the data to the coding software.

**Data Source**
The real-world data comes from UC Irvine Machine Learning Repository's URL Reputation dataset which contains 2.4 million examples and 3.2 million features. Only 160,000 features are used to train the data from Day 0. The link to the original source of data can be found [here]([https://archive-beta.ics.uci.edu/dataset/187/url+reputation]).
